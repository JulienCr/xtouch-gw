# XTouch GW Editor

A SvelteKit single-page app for editing the gateway's `config.yaml` profiles
visually. Built as a static SPA so the Rust gateway can serve it from
`src/api/` at `http://127.0.0.1:<api_port>/editor`.

## Prerequisites

- Node.js 20 or newer
- pnpm 9 or newer (`corepack enable` recommended)

## Commands

```pwsh
pnpm install     # install deps
pnpm dev         # vite dev server on http://localhost:5173
pnpm build       # static build into ./build
pnpm preview     # preview the production build
pnpm check       # svelte-check
```

By default the SPA talks to the gateway at `http://localhost:8125`. To use
canned mock data instead of hitting the real API (the default during early
development), leave `VITE_USE_REAL_API` unset. To switch to the real backend:

```pwsh
$env:VITE_USE_REAL_API = "1"
pnpm dev
```

## Integration with the Rust gateway

The production build emits a static `build/` directory containing
`index.html` plus hashed assets. The Rust side embeds it via `rust-embed` (see
the build integration section in `plans/i-would-like-to-happy-feather.md`)
and serves it at `/editor`.

In dev, the gateway redirects `/editor` to `http://localhost:5173` so this
Vite dev server provides hot-reload while the gateway runs separately.

## Layout

```
src/
  app.html
  app.css                 tailwind entry
  lib/
    api.ts                REST + WS client (mocks during early dev)
    schema.ts             Ajv-backed schema loader
    generated/
      config.schema.json  generated by `cargo run --bin export-schema`
    stores/
      profile.ts          current profile YAML, dirty state, mtime hash
      live.ts             last-touched control, axes, connections
      pickers.ts          OBS scenes/sources, MIDI ports
    components/
      XTouchSurface.svelte
      GamepadSurface.svelte
      ControlList.svelte
      MappingEditor.svelte
      CaptureButton.svelte
      SettingsForm.svelte
      PickerField.svelte
      ProfilesPanel.svelte
      HistoryPanel.svelte
  routes/
    +layout.svelte        nav: Surface | List | Settings | Profiles | History
    +layout.ts            ssr=false, prerender=false (SPA mode)
    +page.svelte          redirects to /surface
    surface/+page.svelte
    list/+page.svelte
    settings/+page.svelte
    profiles/+page.svelte
    history/+page.svelte
static/
  xtouch.svg
  gamepad-faceoff.svg
  gamepad-xbox.svg
```
